## 现在开始吧!

克隆到本地

```shell
$ git clone https://github.com/Ares-Chang/wooden-fish.git
$ cd wooden-fish
$ npm install
$ npm run dev
```

**强推一个小工具 [ni](https://github.com/antfu/ni) ，用上之后爱不释手。**

```shell
$ git clone git@github.com:liuxiyuan-2022/EWoodenFish.git
$ cd wooden-fish
$ ni
$ nr dev
```
